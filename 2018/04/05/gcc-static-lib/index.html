gcc静态链接库的生成和使用

Date: April 5, 2018  19:17:06 Category: 工具使用

创建静态链接库：

假设要将多个源文件s1.c s2.c s3.c打包成静态库文件（在linux下为.a后缀），header.h为声明了文件中函数的头文件

gcc -c s1.c s2.c s3.c

此时生成了s1.o s2.o s3.o

ar -crv libtest.a s1.o s2.o s3.o

静态库的名字前面通常包含lib

使用静态链接库:

在要使用的文件test.c中#include "header.h"

然后编译

gcc -o test test.c -L. -ltest

-L指定了静态库所在路径，.即为当前路径，-l指定静态库的名称去掉lib的部分。中间没有空格

查看静态链接库的文件：

ar -tv libtest.a

解压静态链接库：

ar -x libtest.a
