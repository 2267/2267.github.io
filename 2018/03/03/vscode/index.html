vscode配置记录

Date: March 3, 2018  12:13:18 Category: 工具使用

VSCode功能强大，且轻量，简单配置后即可方便使用

字符集问题

由于Jetbrain的工具电脑带起来比较吃力，打算使用VSCode来进行编程相关的事情。

安装VSCode, code runner插件，工具推荐的有关C/C++，Java，Python运行的插件。

对于字符集这个问题，由于经常使用CMD，通过注册表把CMD加到右键菜单，为了防止以后编码出现问题，使用chcp将CMD的字符集改成了utf-8，然后麻烦就来了。Jdk对于每个系统语言环境设置了不同的字符集，对我来说也就是gbk，经过一番折腾通过在系统变量加入JAVA_TOOL_OPTIONS:-Dfile.encoding=UTF-8可以解决这个问题。

对于python代码在VSCode里面输出中文乱码的问题，通过加入系统变量PYTHONIOENCODING=UTF8解决。

对于C/C++，配置起来比较麻烦。实际上使用code runner可以直接运行，不过插件功能不能工作。需要修改.json文件。暂时不打算解决。

wsl颜色问题

更新1809后，在vscode中打开wsl发现颜色出现了问题（输入输出位置反色）
由于我本人喜欢白色主题，故cmd和vscode都是白底黑字的
这个问题可以通过将cmd的默认色改回原来的黑底白字再改回去解决

同步

  • 安装插件Settings Sync
  • 在github上获取access token
  • 将此token输入到setting-sync中
  • ctrl-p sync进行同步、查看相关命令

目前用到的插件

  • C/C++
  • Settings
  • LaTex Workshop
  • Markdown All in One
  • Prettire - Code formatter

VSCode十分给力，Typora和TexStudio都可以抛弃了
