有关重装系统

Date: March 2, 2018  15:54:36 Category: 操作系统

最早开始装系统的时候，使用一键ghost这种，虽然不懂但是还是跟着网上的教程重装了下来，然而至今对这种gho文件不了解，也不知道PE装系统是什么东西，也很不理解为什么网上对装系统玩了这么多的花样。大概唯一合理的解释就是做系统的人在系统文件中装了推广，或者在系统后台弄了什么常驻程序，刷流量等等，或者是当肉鸡，偷隐私？反正他们对系统想做什么就做什么。

之后学会了用光盘烧录工具在u盘做启动盘。这种做法应该说是比较通用的。选择的系统原盘都会从“我告诉你”上面下载，大家公认这个网站保存了windows原版镜像，对于linux直接进官网就能下载。做好了启动盘以后，在bios修改启动顺序，或者开机到进系统间按某个键，选择从u盘启动。工具的作用无非就是让u盘里的文件具有启动功能，可以用电脑直接在u盘启动。这个做法是正常的做法。

最近做了win7启动盘，从setup.exe这个文件中冒出了一个大胆的想法，由此偶然发现windows系统重装实际上很简单。这种简单的做法适用于windows重装位数相同windows，实际上从网上下载来的系统原盘iso，直接解压就能得到和启动盘里面差不多的文件，唯一不同的是不具备直接启动的功能。将解压后的文件放到非系统盘中，运行setup.exe，不要格式化这个文件所在的分区，就可以简单的重装系统了。

所以这种做法的要满足的条件是：

  • 原来是windows，要重装的也是windows
  • 原来是32（或64）位，要重装32（或64）位
  • 电脑中不只是有一个分区（除了系统所在分区还有分区）
