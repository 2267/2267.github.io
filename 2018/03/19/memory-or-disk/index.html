缓存？内存？外存？主存？辅存？硬盘？

Date: March 19, 2018  21:32:34 Category: 体系结构

发现其实很多人对这些词混淆了概念，总结一下：

  • 缓存：这个没什么可说的，cache，位于cpu片上，目前的cpu一般是三级缓存

  • 主存/内存/一级存储：英文为main/primary/internal memory，指电脑中的内存条，或者手机上的运行内存，也就是一般在1g-16g之间的这个数值，在嵌入式系统中可能是几m到几百m

  • 辅存/外存/二级存储：英文auxiliary/second/external memory，指电脑中的硬盘/磁盘/固态硬盘，手机中的闪存等等，一般大小在几百g到几t。以及U盘，移动硬盘，网盘等等

  • 内存：由于硬盘和主存都有memory，在翻译的时候很多人将memory都翻译为内存，所以大家说的内存既包括主存又包括硬盘

    ​

此外

  • ROM：Read Only Memory。其实现在的硬盘并不是只读的，是可写可读的。实际上正确的描述是非易的存储器（NVM）即断电之后仍保留数据
  • RAM：Random-Access Memory。实际上硬盘也是随机存储的。RAM其实最开始的正确描述是RWM（read and write memory），但是当时的人觉得RWM没办法发音，所以就用RAM这个词，结果这个词后来又用来描述随机访存，结果就混乱了。
  • 目前的cache使用SRAM（貌似第三集缓存是一种特殊的DRAM）
  • 目前的主存是建立在DRAM之上的，例如DDR3
  • 硬盘/磁盘/机械硬盘：使用磁头和盘片来存取数据
  • Flash：是建立在EEPROM上的，用于USB
  • SSD：是建立在Flash上的，用于固态硬盘
