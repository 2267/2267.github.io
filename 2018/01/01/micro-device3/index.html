微电子器件笔记-3

Date: January 1, 2018  0:00:00 Category: 集成电路

MOSFET：

场效应晶体管FET优点：

1.
输入阻抗高，有利于各级间的直接耦合，有利于在大功率晶体管中将各子晶体管并接，有利于输入端与微波系统的匹配。

2. 温度稳定性好

3. 噪声较小

4. 在大电流情况下跨导基本不降

5. 没有少子存储效应，因此开关速度快

6. 功耗低

7. 制造工艺简单

8. 各管之间存在天然隔离，适宜于制作VLSI

FET分为三类：

结型场效应晶体管JFET：利用PN结为控制栅

肖特基势垒场效应晶体管MESFET：利用肖特基势垒结为控制栅

绝缘栅型场效应晶体管IGFET/金属-氧化物-半导体场效应管MOSFET

电子/空穴在栅极附近的硅表面便形成了一个薄层，称之为反型层，反型层比下面的耗尽层要薄的多

当硅表面处少子浓度达到或超过体内平衡多子浓度时，称为表面发生了强反型

强反型时，表面势近似为不变的数值，耗尽层电荷及耗尽层厚度有极大值，此时过剩栅压只是形成反型层电荷

使栅下硅表面处开始发生强反型时的电压称为阈电压或开启电压

MOS结构：

理想情况：费米势：
$\phi_{\text{Fp}} = \frac{kT}{q}\ln\frac{N_{A}}{n_{i}} > 0$
$\phi_{\text{Fn}} = - \frac{kT}{q}\ln\frac{N_{D}}{n_{i}} < 0$

实际情况：$金属半导体间的功函数差\phi_{\text{MS}} < 0\ ，栅氧化层电荷面密度\ Q_{\text{OX}} > 0$，使半导体一侧带负电荷

表面势：$\phi_{S} = - \phi_{\text{MS}} + \frac{Q_{\text{OX}}}{C_{\text{OX}}}$，为能带的弯曲量除以q，其中$C_{\text{OX}} = \frac{\varepsilon_{\text{OX}}}{T_{\text{OX}}}$

VG=平带电压VFB：使半导体表面势恢复为零
，$V_{\text{FB}} = \phi_{\text{MS}} - \frac{Q_{\text{OX}}}{C_{\text{OX}}}$，是一个负值

VG=VT时：$V_{G} - V_{\text{FB}} = V_{\text{OX}} + \phi_{S}$
，发生强反型的表面势$\phi_{S，\text{inv}} = 2\phi_{\text{FB}}$

$V_{T} = V_{\text{FB}} + V_{\text{OX}} + 2\phi_{\text{FB}}$
其中$V_{\text{OX}} = \frac{- Q_{S}}{C_{\text{OX}}}$，QS=Qn+QA$\approx$QA

半导体中的电荷面密度=反型层中的电子电荷面密度+耗尽区中的电离受主电荷面密度

QA是表面势的函数

$V_{T} = \phi_{\text{MS}} - \frac{Q_{\text{OX}}}{C_{\text{OX}}} - \frac{Q_{A}(2\phi_{\text{Fp}})}{C_{\text{OX}}} + 2\phi_{\text{Fp}}$

MOSFET结构：体因子$\mathbf{K}$

N沟道：$V_{T} = V_{S} + \phi_{\text{MS}} - \frac{Q_{\text{OX}}}{C_{\text{OX}}} + K{\lbrack 2\phi_{\text{Fp}} + V_{S} - V_{B}\rbrack}^{\frac{1}{2}} + 2\phi_{\text{Fp}}$
P型衬底：$K = \frac{ {(2q\varepsilon_{s}N_{A})}^{1/2} }{C_{\text{OX} } }$

P沟道：$V_{T} = V_{S} + \phi_{\text{MS}} - \frac{Q_{\text{OX}}}{C_{\text{OX}}} - K{\lbrack - (2\phi_{\text{Fn}} + V_{S} - V_{B})\rbrack}^{\frac{1}{2}} + 2\phi_{\text{Fn}}$
N型衬底：$K = \frac{ {(2q\varepsilon_{s}N_{D})}^{1/2} }{C_{\text{OX} } }$

影响阈电压的因素：

1. 栅氧化层厚度TOX：TOX越薄，|VT|越小

2.
衬底费米势$\phi_{\text{FB}}$：与掺杂浓度有关，但影响不大，绝对值的典型值为0.3V

3.
功函数差$\phi_{\text{MS}}$：与金属种类、半导体导电类型及掺杂浓度有关，典型值为-0.6V-$\phi_{\text{FB}}$

4. 耗尽区电离杂质电荷面密度：近似QAD∝NAD^1/2^

N沟：$Q_{A} = - qN_{Ax_{d}} = - {(4qN_{A}\varepsilon_{s}\phi_{\text{Fp}})}^{1/2}$<0

P沟：$Q_{D} = qN_{Dx_{d}} = {( - 4qN_{D}\varepsilon_{s}\phi_{Fn})}^{1/2}$>0

5. 栅氧化层中的电荷面密度QOX：与制造工艺及晶向有关，和氧化以后的工艺

调整阈电压主要是通过改变掺杂浓度N和改变栅氧化层厚度TOX来实现

离子注入对阈电压的影响：注入的杂质与衬底杂质类型相同时，N沟道MOSFET阈电压向正方向调整

衬底偏置效应（体效应）：VT随VBS的变化而变化，VBS绝对值增大时，NMOS阈电压向正方向变化，PMOS阈电压向负方向变化。TOX越厚、N越高，衬底偏置效应就越严重

阈电压的测量：

1. 联立方程法

2. $\sqrt{I_{\text{Dsat}}} - V_{\text{GS}}$法：横坐标截距为VT

3. 1μA法：根据宽长比调整测试电流IDT

直流电流电压方程：增益因子β=$\frac{\mathbf{Z}}{\mathbf{L}}\mathbf{\mu}{\mathbf{n}}\mathbf{C}{\mathbf{\text{OX}}}$

非饱和区的简化：萨之唐方程：$I_{D} = \beta(V_{\text{GS}} - V_{T} - \frac{1}{2}V_{\text{DS}})V_{\text{DS}}$

饱和漏极电流：$I_{D} = \frac{\beta}{2}{(V_{\text{GS}} - V_{T})}^{2}$

饱和漏极电压：VDsat=VGS-VT

有效沟道长度调制效应：沟道有效长度随VDS的增大而缩短的现象

漏区静电场对沟道的反馈作用：耗尽区宽度接近有效沟道长度，漏区的电力线有一部分终止于沟道上，使沟道电子的数量增多导致电流增大。

衬底为中等或较高掺杂时有效沟道长度调制效应占主导，掺杂浓度低时漏区与沟道间的经典耦合为主

亚阈区导电：

弱反型：表面势位于$\phi_{\text{Fp}}$和$2\phi_{\text{Fp}}$之间

亚阈漏极电流（次开启电流）IDsub：

亚阈区IDsub和V­GS呈指数关系

VDS=0时IDsub=0；VDS较小时IDsub随VDS增大而增大；VDS≫kT/q后,IDsub与VDS无关

亚阈区栅源电压摆幅S：$S \equiv \frac{dV_{\text{GS}}}{d(lnI_{\text{Dsub}})} = \frac{\text{nkT}}{q} = \frac{\text{kT}}{q}(1 + \frac{C_{D}}{C_{\text{OX}}})$
CD是沟道下的耗尽层电容

S的意义：使IDsub扩大e倍所需要的栅源电压VGS的增量，代表亚阈区VGS对IDsub的控制能力，S越大控制能力越弱。作为开关管要求S尽量小，S极限值为kT/q，Smin=26mV
(/e倍)

MOSFET的直流参数：

1. 阈电压VT

2.
饱和漏极电流$\mathbf{I}{\mathbf{\text{DSS}}} = \pm \frac{Z}{2L}\mu C{\text{OX}}V_{T}^{2}$：耗尽型MOSFET当VGS=0，VDS足够大且恒定的漏极电流饱和值

3.
截止漏极电流：只适用于增强型MOSFET，当VGS=0，外加VDS后的亚阈电流与PN结反向电流引起的微小电流

4.
导通电阻Ron=$\frac{1}{\beta(V_{\text{GS}} - V_{T})}$：MOSFET工作于线性区且VDS­­很小的沟道电阻

5. 栅极电流IG：表示从栅极穿过栅氧化层到沟道之间的电流。

温度特性：

阈电压与温度的关系：NMOS阈电压具有负温系数，PMOS阈电压具有正温系数。|$\frac{dV_{T}}{dT}$|大约为每度几个mV，

VT的温度系数的绝对值随|VBS|增加而减小

漏极电流与温度的关系：总的来说，MOSFET有较好的温度稳定性

VGS-VT较大时，漏极电流温度系数$\frac{dI_{D}}{dT} > 0$

VGS-VT较小时，$\ \frac{dI_{D}}{dT} < 0$

VGS-VT$= \frac{\mu(dV_{T}/dT)}{(d\mu_{n}/dT)} + \frac{1}{2}V_{\text{DS}}$时，$\frac{dI_{D}}{dT} = 0$

其他：

亚阈区摆幅S变大

泄露电流变大

迁移率降低，跨导降低

金属线电阻变大

击穿电压：

1. 源漏击穿电压BVDS：

漏PN结雪崩击穿：由于源栅之间存在电场，此击穿电压远低于相同掺杂和结深的PN结

源、漏之间的穿通Vpt：L越小，NA­越小，穿通电压就越低

2. 栅源击穿电压BVGS

小信号交流参数：

1.
跨导gm：${\frac{\partial I_{D}}{\partial V_{\text{GS}}}|}{V{\text{DS}} = 常数}$，栅源电压对漏电流的控制能力

非饱和区：gm=βVDS\ ­饱和区：gm=β(VGS-VT)

2.
漏源电导gds：${\frac{\partial I_{D}}{\partial V_{\text{DS}}}|}{V{\text{GS}} = 常数}$，漏极电流随漏源电压的变化而变化的情况

非饱和区：gds=β(VGS-VT-VDS)，VDS很小时gds=β(VGS-VT)=1/Ron
饱和区(gds)sat=0，实际略大于0

3.
电压放大系数μ：${- \frac{\partial V_{DS}}{\partial V_{\text{GS}}}|}{I{D}}$=$\frac{g_{m}}{g_{\text{ds}}} = \frac{V_{\text{DS}}}{V_{GS} - V_{T} - V_{\text{DS}}}$
饱和区：μs→$\infty$实际上为有效值，希望μs尽量大

小信号高频等效电路：

跨导的截止频率fgm­：代表跨导|gm(ω)|下降到低频值的$\frac{1}{\sqrt{2}}$
