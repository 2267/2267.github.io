微电子器件笔记-1

Date: January 1, 2018  0:00:00 Category: 集成电路

当时为了喜欢的人备考某大学时自学的课程的笔记。公式语法在hexo主题上可能显示的不好。

PN结：

突变结与线性缓变结的定义

理想突变结：P型区和N型区的掺杂浓度NA和ND都是均匀分布的，杂质浓度在结面处发生阶跃式的突变

单边突变结：突变结某一侧的掺杂浓度远大于另一侧

缓变结：P区和N区交界处的杂质分布不是突然低从均匀的受主浓度突变到均匀的施主浓度，而是逐渐变化的

线性缓变结：冶金结附近的杂质浓度是随距离进行线性变化的

PN结空间电荷区的形成

耗尽近似与中性近似

耗尽近似：假设空间电荷区内的自由载流子已完全耗尽，电离杂质构成空间电荷区内电荷的唯一来源

中性近似：认为耗尽区以外区域中的多子浓度仍等于电离杂质浓度，因此这部分区域保持了完全的电中性，采用中性近似时这部分区域可称为中性区

耗尽区宽度，内建电场，内建电势的计算

内建电场：

N区：$\frac{\text{dE}}{\text{dx}} = \frac{q}{\varepsilon_{s}}N_{D}\text{\ \ \ \ \ \ \ \ }E\left( x \right) = \frac{q}{\varepsilon_{s}}N_{D}\left( x - x_{n} \right)\text{\ \ \ \ \ \ \ \ \ \ \ \ \ }0 \leq x \leq x_{n}$

P区：$\frac{\text{dE}}{\text{dx}} = - \frac{q}{\varepsilon_{s}}N_{A}\text{\ \ \ \ \ \ \ \ }E\left( x \right) = - \frac{q}{\varepsilon_{s}}N_{A}\left( x + x_{p} \right)\text{\ \ \ \ \ } - x_{p} \leq x \leq 0$

内建电场的方向是从右侧的N区指向左侧的P区，为负值

耗尽区宽度：

$\left| E \right|{\max} = \left| E\left( 0 \right) \right| = \frac{q}{\varepsilon{s}}N_{D}x_{n} = \frac{q}{\varepsilon_{s}}N_{A}x_{p}$
$qN_{D}x_{n} = qN_{A}x_{p}$

$x_{n} = \frac{\varepsilon_{s}}{qN_{D}}\left| E \right|{\max}$
$x{p} = \frac{\varepsilon_{s}}{qN_{A}}\left| E \right|_{\max}$

$x_{d} = \frac{\varepsilon_{s}}{qN_{0}}\left| E \right|{\max} = \frac{2V{\text{bi}}}{\left| E \right|{\max}}$
$N{0} = \frac{N_{A}N_{D}\ }{N_{A}{+ N}_{D}}$，称为约化浓度

N区，P区耗尽区中电荷量大小相等，掺杂浓度越高，耗尽区就越薄

内建电势/扩散电势：

$V_{\text{bi}} = \frac{\varepsilon_{s}}{2qN_{0}}{\left| E \right|{\max}}^{2}$
$\left| E \right|{\max} = {(\frac{2qN_{0}}{\varepsilon_{s}}V_{\text{bi}})}^{1/2}$

$V_{\text{bi}} = \frac{\text{kT}}{q}\ln\frac{N_{A}N_{D}}{n_{i}^{2}}$

对于单边突变结的简化：

P^+^N：NA≫N­D N0$\approx$ND $x_{p} \approx 0$

PN^+^：ND≫N­A N0$\approx$NA $x_{n} \approx 0$

耗尽区主要分布在低掺杂的一侧，$\left| E \right|{\max}$和$x{d}$也主要取决于低掺杂一侧的杂质浓度

对于线性缓变结：

$N_{D} - N_{A} = ax$

$\frac{\text{dE}}{\text{dx}} = \frac{q}{\varepsilon_{s}}{(N}{D} - N{A}) = \frac{q}{\varepsilon_{s}}\text{ax}$

$\left| E\left( x \right) \right| = \left| E_{\max} \right|\lbrack 1 - {(\frac{2x}{x_{d}})}^{2}\rbrack$
$\left| E_{\max} \right| = \frac{\text{aq}x_{d}^{2}}{8\varepsilon_{s}}$

$V_{\text{bi}} = \frac{\text{kT}}{q}\ln\frac{N_{A}N_{D}}{n_{i}^{2}} = \frac{2\text{kT}}{q}\ln\frac{ax_{d}}{2n_{i}} = \frac{2}{3}\left| E_{\max} \right|x_{d}$

其中$N_{A}\left( - \frac{x_{d}}{2} \right) = N_{D}\left( \frac{x_{d}}{2} \right) = \frac{ax_{d}}{2}$

PN结的能带图

电位分布$\Psi(x)$乘以-q得到电位能分布

载流子浓度分布

势垒区内及边界：

$n = n_{i}exp( - \frac{E_{i} - E_{F}}{\text{kT}})$
$p = n_{i}exp( - \frac{E_{F} - E_{i}}{\text{kT}})$

$E_{i}\left( x \right) = E_{\text{in}} - q\Psi\left( x \right) = E_{\text{ip}} - q\Psi\left( x \right) - qV_{\text{bi}}$

$n\left( x \right) = n_{n0}exp(\frac{q\Psi(x)}{\text{kT}})$
$p\left( x \right) = p_{p0}exp( - \frac{qV_{\text{bi}} + q\Psi(x)}{\text{kT}})$

$n\left( x \right)p\left( x \right) = n_{n0}p_{p0}exp( - \frac{qV_{\text{bi}}}{\text{kT}})$

$V_{\text{bi}} = \frac{\text{kT}}{q}\ln\frac{n_{n0}p_{p0}}{n_{i}^{2}} = \frac{\text{kT}}{q}\ln\frac{N_{A}N_{D}}{n_{i}^{2}}$

$x = - x_{p}处：\Psi\left( x \right) = - V_{\text{bi}}\text{\ \ \ n}\left( x \right) = n_{p0} = n_{n0}exp( - \frac{qV_{\text{bi}}}{\text{kT}})\ \ \ \ \ p\left( x \right) = p_{p0}$

$x = x_{n}处：\text{\ \ \ }\Psi\left( x \right) = 0\ \ \ \ \ \ \ \ \ n\left( x \right) = n_{n0}\text{\ \ \ \ \ p}\left( x \right) = p_{n0} = p_{p0}exp( - \frac{qV_{\text{bi}}}{\text{kT}})\ $

设外加电压全部落在耗尽区上，只需将公式中的$V_{\text{bi}}$换成$(V_{\text{bi}} - V)$

结定律：PN结有外加电压V时，边界处的少子等于平衡时的少子浓度乘以
exp($\frac{\text{qV}}{\text{kT}})$，对正反电压都适用，但正向时只适用于小注入

$p_{n}\left( x_{n} \right) = p_{n0}\exp\left( \frac{\text{qV}}{\text{kT}} \right)\text{\ \ \ \ }$
$n_{p}\left( - x_{p} \right) = n_{p0}\exp\left( \frac{\text{qV}}{\text{kT}} \right)\text{\ \ }$

正向及反向电压下PN结中的载流子运动情况

掺杂的浓度不变时，电场的斜率应不变

外加正向电压后，$x_{d}$与$\left| E_{\max} \right|$减小，PN结势垒高度降低为$q(V_{\text{bi}} - V)$

势垒高度降低后不能再阻止N区电荷向P区扩散及P区空穴向N区扩散，于是形成正向电流。由于正向电流的电荷来源是多子，所以正向电流很大

正向电流密度由三部分组成：

1. 空穴扩散电流密度Jdp

2. 电子扩散电流密度Jdn

3. 势垒区复合电流密度Jr

外加反向电压后，$x_{d}$与$\left| E_{\max} \right|$增大，PN结势垒高度增高到$q\left( V_{\text{bi}} - V \right)\ \ \ \ \ \ \ (V < 0)$

多子面临的势垒提高了，更不能扩散到对方区域中去了，但少子面临的势阱也更深了，所以更容易被反向电场拉入对方区域，从而形成反向电流。由于反向电流的电荷来源是少子，所以反向电流很小

反向电流密度由三部分组成：

1. 空穴扩散电流密度Jdp

2. 电子扩散电流密度Jdn

3. 势垒区产生电流密度Jg

PN结的少子分布图

少子浓度的边界条件：

N区：
$p_{n}\left( x_{n} \right) = p_{n0}\exp\left( \frac{\text{qV}}{\text{kT}} \right)\text{\ \ \ \ \ }p_{n}|{x \rightarrow \infty} = p{n0}$

P区：
$n_{p}\left( - x_{p} \right) = n_{p0}\exp\left( \frac{\text{qV}}{\text{kT}} \right)\text{\ \ \ \ \ }n_{p}|{x \rightarrow - \infty} = n{p0}$

对于非平衡少子：

N区：
$\mathrm{\Delta}p_{n}\left( x_{n} \right) = p_{n0}\lbrack\exp{\left( \frac{\text{qV}}{\text{kT}} \right) - 1\rbrack}\ \ \ \ \ \mathrm{\Delta}p_{n}|_{x \rightarrow \infty} = 0$

P区：
$\mathrm{\Delta}n_{p}\left( - x_{p} \right) = n_{p0}\lbrack\exp{\left( \frac{\text{qV}}{\text{kT}} \right) - 1\rbrack}\ \ \ \ \ \mathrm{\Delta}n_{p}|_{x \rightarrow - \infty} = 0$

中性区：足够长远大于扩散长度 $L_{p} = \sqrt{D_{p}\tau_{p}}$
$L_{n} = \sqrt{D_{n}\tau_{n}}$

N区：
$\mathrm{\Delta}p_{n}\left( x_{n} \right) = p_{n0}\lbrack\exp{\left( \frac{\text{qV}}{\text{kT}} \right) - 1\rbrack}exp( - \frac{x - x_{n}}{L_{p}})$

P区：
$\mathrm{\Delta}n_{p}\left( - x_{p} \right) = n_{p0}\lbrack\exp{\left( \frac{\text{qV}}{\text{kT}} \right) - 1\rbrack}exp(\frac{x + x_{p}}{L_{n}})$

经过一个L的长度，非平衡载流子浓度将为1/e

PN结的直流伏安特性

N区内空穴扩散电流密度：$J_{\text{dp}} = \frac{qD_{p}p_{n0}}{L_{p}}\lbrack\exp{\left( \frac{\text{qV}}{\text{kT}} \right) - 1\rbrack}$

P区内电子扩散电流密度：$J_{dn} = \frac{qD_{n}n_{p0}}{L_{n}}\lbrack\exp{\left( \frac{\text{qV}}{\text{kT}} \right) - 1\rbrack}$

PN结总扩散电流密度：肖克莱方程（理想二极管的电流-电压方程）

$J_{d} = J_{\text{dp}} + J_{\text{dn}} = J_{0}\lbrack\exp{\left( \frac{\text{qV}}{\text{kT}} \right) - 1\rbrack}\text{\ \ \ }$

$\ J_{0} = q\left( \frac{D_{p}}{L_{p}}p_{n0} + \frac{D_{n}}{L_{n}}n_{p0} \right) = qn_{i}^{2}(\frac{D_{p}}{L_{p}N_{D}} + \frac{D_{n}}{L_{n}N_{A}})$

$当V = 0时，J_{d} = 0$

$当V \gg \frac{\text{kT}}{q}时，J_{d} = J_{0}\exp\left( \frac{\text{qV}}{\text{kT}} \right)$

$当V < 0且\left| V \right| \gg \frac{\text{kT}}{q}时，J_{d} = - J_{0}$

PN结反向饱和电流的计算及影响因素

$反向饱和电流：I_{0} = AJ_{0}$ A：PN结的结面积

近似：

1. 突变耗尽层近似

2. 玻尔兹曼统计近似

3. 小注入假设

4. 忽略耗尽区内的产生-复合电流，并假设耗尽区内电子电流和空穴电流恒定

影响J0的因素：

与材料种类的关系：$E_{G} \uparrow ，则n_{i} \downarrow ，J_{0} \downarrow$

与掺杂浓度的关系：$N_{D}、N_{A} \uparrow ，则p_{n0}、n_{p0} \downarrow ，J_{0} \downarrow$，主要取决于低掺杂一侧的杂质浓度

与温度T的关系：$T \uparrow ，则n_{i} \uparrow ，J_{0} \uparrow$，因此$J_{0}$具有正温度系数，这是影响PN结热稳定性的重要因素

势垒区产生复合电流Jgr：

$J_{\text{gr}} = q\int_{- x_{p}}^{x_{n}}\text{Udx}$
，净复合率$U = \frac{\text{np} - n_{i}^{2}}{\tau(n + p + 2n_{i})}$
P区内：$U \approx \frac{\Delta n}{\tau_{n}}$
N区内：$U \approx \frac{\Delta p}{\tau_{p}}$

外加电压时 $\text{np} = n_{i}^{2}exp(\frac{\text{qV}}{\text{kT}})$

$当V = 0时$，$\text{np} = n_{i}^{2}$，U=0

$当V > 0时$，$\text{np} > n_{i}^{2}$，U>0，发生净复合

$当V < 0时$，$\text{np} < n_{i}^{2}$，U<0，发生净产生

简化计算，势垒区内：

$J_{\text{gr}} = qUx_{d} = \frac{qn_{i}x_{d}}{2\tau}\frac{\exp\left( \frac{\text{qV}}{\text{kT}} \right) - 1}{\exp\left( \frac{\text{qV}}{2kT} \right) - 1}$

$当V = 0时$，Jgr=0

$当V \gg \frac{\text{kT}}{q}时$，$J_{r} \approx \frac{qn_{i}x_{d}}{2\tau}\exp\left( \frac{\text{qV}}{2kT} \right)$

$当V < 0且\left| V \right| \gg \frac{\text{kT}}{q}时，J_{g} \approx - \frac{qn_{i}x_{d}}{2\tau}$

温度一定时，外加正向电压且$V \gg \frac{\text{kT}}{q}$：

V比较小时，以Jr为主，V比较大时，以Jd为主，EG越大，则过渡电压越高

外加反向电压且$|V| \gg \frac{\text{kT}}{q}$：

温度较低时，以Jg为主，温度较高时，以Jd为主，EG越大，则过渡温度越高

正向导通电压VF：

影响正向导通电压的因素：和J0相反，其中EG是影响最大的因素

薄基区二极管的特点

中性区的长度小于少子扩散长度，Jd­­会因为少子浓度的边界条件不同而有所不同，但势垒区Jgr的表达式无变化

$\mathrm{\Delta}p_{n}\left( x_{n} \right) = p_{n0}\lbrack\exp{\left( \frac{\text{qV}}{\text{kT}} \right) - 1\rbrack}(1 - \frac{x}{W_{B}})$

空穴扩散电流密度：$J_{\text{dp}} = \frac{qD_{p}p_{n0}}{W_{B}}\lbrack\exp{\left( \frac{\text{qV}}{\text{kT}} \right) - 1\rbrack}$，$W_{B}$替换$L_{p}$

电子扩散电流密度：$J_{dn} = \frac{qD_{n}n_{p0}}{W_{E}}\lbrack\exp{\left( \frac{\text{qV}}{\text{kT}} \right) - 1\rbrack}$，$W_{E}$替换$L_{n}$

大注入效应

小注入条件：注入某区边界附近的非平衡少子浓度远小于该区的平衡多子浓度

大注入条件：注入某区边界附近的非平衡少子浓度远大于该区的平衡多子浓度

大注入下的结定律：$p_{n}\left( x_{n} \right) = n_{i}\exp\left( \frac{\text{qV}}{2\text{kT}} \right)\text{\ \ \ \ }$
$n_{p}\left( - x_{p} \right) = n_{i}\exp\left( \frac{\text{qV}}{2\text{kT}} \right)\text{\ \ }$

大注入下的自建电场：$E(x) = - \frac{D_{n}}{\mu_{n}}\frac{1}{n_{n}}\frac{dn_{n}}{\text{dx}} = - - \frac{D_{p}}{\mu_{p}}\frac{1}{p_{n}}\frac{dp_{n}}{\text{dx}}$

大注入条件下的PN结电流：

$J_{p} = - q(2D_{p}\frac{dp_{n}}{\text{dx}})$
相当于空穴电流仍只由扩散电流组成，但扩散系数扩大了一倍，这个现象称为韦伯斯脱效应

$J_{p} = \frac{\sqrt{2}qD_{p}n_{i}}{L_{p}}\exp\left( \frac{\text{qV}}{2\text{kT}} \right)$
$J_{n} = \frac{\sqrt{2}qD_{n}n_{i}}{L_{n}}\exp\left( \frac{\text{qV}}{2\text{kT}} \right)$

转折电压/膝电压：注入程度的大小取决于外加正向电压的大小，Vk为从小注入过渡到大注入的转折电压

$V_{\text{KN}} = \frac{2kT}{q}ln(\frac{\sqrt{2}N_{D}}{n_{i}})$
$V_{\text{KP}} = \frac{2kT}{q}ln(\frac{\sqrt{2}N_{A}}{n_{i}})$

PN结击穿

击穿机理有三种：电击穿（雪崩倍增、隧道效应）、热击穿

雪崩击穿：

碰撞电离率αin/αip：一个自由电子（或空穴）在单位距离内通过碰撞电离产生的新的电子空穴对的数目，与电场强烈有关，近似表达式为$\alpha_{i} = Aexp\lbrack - {(\frac{B}{E})}^{m}\rbrack$

雪崩倍增因子：$M = \frac{1}{1 - \int_{0}^{x_{d}}{\alpha_{i}\text{dx}}}$，包括碰撞电离作用在内的流出耗尽层的总电流与未发生碰撞电离时的原始电流之比

雪崩击穿条件：$\int_{0}^{x_{d}}{\alpha_{i}\text{dx}} \rightarrow 1$

雪崩击穿电压的近似计算：当Emax达到某临界电场EC时，即满足击穿条件，Ec与结的典型值为Ec=2*10^5^V/cm

突变结：$V_{B} = \frac{\varepsilon_{s}}{2qN_{0}}{E_{c}}^{2}$
线性缓变结：$V_{B} = {(\frac{32\varepsilon_{s}}{9aq})}^{1/2}{E_{c}}^{3/2}$

室温下：单边突变结：$V_{B} \approx 5.2*10^{13}{E_{G}}^{3/2}{N_{0}}^{- 3/4}$
线性缓变结：$V_{B} \approx 10^{10}{E_{G}}^{6/5}a^{- 2/5}$

影响因素：N0，a，T

齐纳击穿的机理及特点、热击穿的机理

提高雪崩击穿电压的措施：

掺杂浓度要低、浓度梯度要小

低掺杂区的厚度要足够厚

结深要深

采用台面结构

齐纳击穿：

隧道效应：由于电子具有波动性，可以有一定的几率穿过势垒。势垒越薄，隧道效应就越明显

齐纳击穿条件：$d_{\min} = \frac{E_{G}}{q{|E}_{\max}|}$足够小

随着反向电压的提高，${|E}{\max}|$增大，隧道长度d缩短，使得反向电流增大。当反向电压增大到使${|E}{\max}|$达到临界值时，d足够小，使反向电流急剧增大，这种现象就称为齐纳击穿或隧道击穿。

热击穿

PN结势垒电容与扩散电容的定义、计算与特点

势垒电容CT：

外加电压有-ΔV的变化时，势垒区宽度发生变化，使势垒区中的空间电荷也发生相应的ΔQ的变化，全称为PN结势垒微分电容。

$$C_{T} = \left| \frac{\text{dQ}}{\text{dV}} \right| = A\frac{\varepsilon_{s}}{x_{d}}$$

势垒区中电离杂质电荷随外加电压的变化率

正负电荷在空间上是分离的

与直流偏压成幂函数关系

正偏反偏下均存在，可作电容器使用

$$C_{D} = \frac{qI_{F}\tau}{2kT} = \frac{1}{2}g_{D}\tau$$

中性区中非平衡载流子电荷随外加电压的变化率

正负电荷在空间上是重叠的

与直流电流成线性关系，与直流偏压成指数关系

只存在于正偏下

直流增量电导：$g_{D} = \frac{dI_{F}}{dV_{0}} = \frac{qI_{F}}{\text{kT}}$
交流导纳：$Y = g_{D} + j\omega C_{D}$

PN结的交流小信号参数与等效电路

势垒电容CT，增量电导gD，扩散电容CD

寄生参数：漏电导g1，寄生电阻rs

直流分量：$I_{F} = Aq\left( J_{dp0} + J_{dn0} \right) = Aq(\frac{D_{P}p_{n0}}{L_{p}} + \frac{D_{n}n_{p0}}{L_{n}})exp(\frac{qV_{0}}{\text{kT}})$

交流分量：$I_{1} = I_{F}\frac{qV_{1}}{\text{kT}}{(1 + j\omega\tau)}^{1/2}$

扩散电流：$I = I_{F} + i = I_{F} + I_{1}e^{\text{jωt}}$

PN结的开关特性与少子存储效应

理想开关特性：

直流特性：开电压为0，关电流为0

瞬态特性：打开瞬间立刻出现稳态电流，关断瞬间电流立刻消失

直流：正向时：rs越小越好，反向时：g1越小越好

瞬态：存储时间ts，下降时间tf，反向恢复时间tr=ts+tf，tr持续的过程叫反向恢复过程

$t_{r} = \tau_{p}\ln\left( \frac{I_{r} + I_{r}}{I_{r}} \right)$
