3.半导体中电子的运动 有效质量

对于半导体来说，起作用的常常是接近于能带底部或能带顶部的电子

$E\left( k \right) - E\left( 0 \right) = \frac{\hslash^{2}k^{2}}{2m_{n}^{}}$
$m_{n}^{} = \frac{\hslash^{2}}{\frac{d^{2}E}{dk^{2}}}$

$v = \frac{\text{ℏk}}{m_{n}^{}} = \frac{1}{\hslash}\frac{\text{dE}}{\text{dk}}$
$a = \frac{f}{m_{n}^{}} = \frac{f}{\hslash^{2}}\frac{d^{2}E}{dk^{2}}$

能带顶部电子的有效质量是负值，能带底部的电子有效质量是正值

引进有效质量的意义在于它概括了半导体内部势场的作用，使得在解决半导体中电子在外力作用下的运动规律时，可以不涉及半导体内部势场的作用。特别是有效质量可以直接由实验测定。引入有效质量后可使问题变得简单，直接把外力和电子的加速度联系起来。

二、半导体中杂质和缺陷能级

缺陷分为三类：1.点缺陷，如空位、间隙原子2.线缺陷，如位错3.面缺陷，如层错、多晶体中的晶粒间界等。

极微量的杂质和缺陷能够对半导体材料和物理性质和化学性质产生决定性的影响，也严重地影响半导体器件的质量。目前用于生产一般硅平面器件的硅单晶，要求控制位错密度在1000$\text{cm}^{- 2}$以下。

原因：由于杂质和缺陷的存在，会使严格按周期性排列的原子所产生的周期性势场受到破坏，有可能在禁带中引入允许电子具有的能量状态（即能级）。

即：杂质和缺陷能在禁带中引入能级。

硅、锗晶体中的杂质能级

杂质进入半导体硅以后，只可能以两种方式存在。间隙式杂质和替位式杂质。3、5族在硅、锗晶体中都是替位式杂质。

电子脱离杂质原子的束缚成为导电电子的过程成为杂质电离。使这个多余的价电子挣脱束缚成为导电电子所需要的能量成为杂质电离能。用$\mathrm{\Delta}E_{D}$表示。5族杂质电离时能够释放电子而产生导电电子并形成正电中心，称他们为施主杂质或n型杂质。释放电子的过程叫做施主电离。施主杂质未电离时是中性的，成为束缚态或中性态，电离后成为正电中心，称为离化态。把主要依靠导带电子导电的半导体称为电子型或n型半导体。

3族杂质在硅、锗中能够接受电子而产生导电空穴并形成负电中心，所以称它们为受主杂质或p型杂质。空穴挣脱受主杂质束缚的过程称为受主电离。受主杂质未电离时是中性的，称为束缚态或中性态。电离后成为负电中心，称为受主离化态。把主要依靠空穴导电的半导体称为空穴型或p型半导体。

价带电子激发成为导带电子的过程成为本征激发。

实验证明硅、锗中的3、5族杂质的电离能都很小，所以受主能级很接近于价带顶，施主能级很接近与导带底。通常将这些杂质能级称为浅能级，将产生浅能级的杂质称为浅能级杂质。浅能级杂质电离能很低，可以利用类氢模型来估算杂质的电离能。

氢原子：

$E_{n} = - \frac{m_{0}q^{4}}{2{(4\pi\varepsilon_{0})}^{2}\hslash^{2}\text{\ n}^{2}}$
$E_{0} = 13.6eV$

$\mathrm{\Delta}E_{D} = \frac{m_{n}^{}q^{4}}{2{(4\pi\varepsilon_{0}\varepsilon_{r})}^{2}\hslash^{2}} = \frac{m_{n}^{} }{m_{0}}\frac{E_{0} }{ {\varepsilon_{r} }^{2} }$
$\mathrm{\Delta}E_{A} = \frac{m_{p}^{}q^{4}}{2{(4\pi\varepsilon_{0}\varepsilon_{r})}^{2}\hslash^{2} } = \frac{m_{p}^{}}{m_{0}}\frac{E_{0}}{ {\varepsilon_{r}}^{2} }$

锗、硅的相对介电常数分别为16、12。

类氢模型只是对实际情况的一个近似。

$N_{D}$表示施主杂质浓度，$N_{A}$表示受主杂质浓度

经过杂质补偿后半导体中的净杂质浓度称为有效杂质浓度。$N_{D} \approx N_{A}$时称为杂质的高度补偿，性能很差，不能用来制造半导体器件。

非3、5族杂质在硅、锗的禁带中产生的施主能级距离导带底较远，它们产生的受主能级距离价带顶也较远，通常称为深能级和深能级杂质

这些深能级杂质能够产生多次电离，每一次电离相应地由一个能级，往往能引入若干个能级，而且有的杂质既能引入施主能级又能引入受主能级。

深能级杂质一般情况下含量很少，而且能级较深，它们对半导体中的载流子浓度和导电类型的影响没有浅能级杂质显著，但对于载流子的复合作用比浅能级杂质强，故这些杂质也称为复合中心。

  1. 3、5族化合物中的杂质能级
  1. 氮化镓、氮化铝、碳化硅中的杂质能级
  1. 缺陷、位错能级

点缺陷：

弗仓克耳缺陷：一部分原子会获得足够的能量，克服周围原子对它的束缚，挤入晶格原子间的间隙，形成间隙原子，原来的位置便形成空位。这时间隙原子和空位是成对出现的。

肖特基缺陷：只在晶体内形成空穴而无间隙原子。

这两种由温度决定的点缺陷又称为热缺陷，它们总是同时存在的。

空位比间隙原子多得多，因而空位是常见的点缺陷。

替位原子：这类缺陷在离子性强的化合物中存在的概率很小，所以在离子晶体中常可以忽略它们的作用。这种缺陷也称为反结构缺陷。

位错：

目前仅仅对具有金刚石结构的硅、锗中的位错了解得稍多一些。

三、半导体中载流子的统计分布

半导体的导电性强烈地随温度而变化，这种变化主要是由于半导体中载流子浓度随温度而变化造成的

状态密度

假定在能带中能量E~(E+dE)之间无限小的能量间隔内有dZ个量子态，则状态密度g(E)为

$$g\left( E \right) = \frac{dZ}{\text{dE}}$$

在k空间中，电子的允许量子态密度是$\frac{2V}{8\pi^{3}}$，这时，每个量子态最多只能容纳一个电子。

导带底部附近$\text{\ \ \ \ \ }E\left( k \right) = E_{c} + \frac{\hslash^{2}k^{2}}{2m_{n}^{*}}$

导带底附近状态密度为
$g_{C}\left( E \right) = \frac{V}{2\pi^{2}}\frac{ {(2m_{n}^{*})}^{\frac{3}{2} } }{\hslash^{3}}{(E - E_{c})}^{\frac{1}{2} }$

价带顶附近状态密度为
$g_{V}\left( E \right) = \frac{V}{2\pi^{2} }\frac{ {(2m_{p}^{*})}^{\frac{3}{2} } }{\hslash^{3} }{(E_{v} - E)}^{\frac{1}{2}}$

费米能级和载流子的统计分布

费米分布函数：$f\left( E \right) = \frac{1}{1 + exp(\frac{E - E_{F}}{k_{0}T})}$

空穴的费米分布函数：$1 - f\left( E \right) = \frac{1}{1 + exp(\frac{E_{F} - E}{k_{0}T})}$

描写了热平衡条件下，电子在允许的量子态上如何分布的一个统计分布函数，$k_{0}$是玻耳兹曼常数，T是热力学温度。$E_{F}$是费米能级或费米能量，和温度、半导体材料的导电类型、杂质的含量以及能量零点的选取有关。

当量子态的能量等于费米能级时，则该量子态被电子占据的概率为50%。

玻耳兹曼分布函数：$f_{B}\left( E \right) = exp(\frac{E_{F} - E}{k_{0}T})$

空穴的玻耳兹曼分布函数：$1 - f_{B}\left( E \right) = exp(\frac{E - E_{F}}{k_{0}T})$

在半导体中，通常费米能级位于禁带内，而且与导带底或价带顶的距离远大于$k_{0}T$，一般可以用玻耳兹曼分布函数描写。

服从玻耳兹曼统计律的电子系统称为非兼并系统。

服从费米统计律的电子系统称为简并性系统。

导带中的电子大多数是在导带底附近，价带中大多数空穴则在价带顶附近。

$f\left( E_{C} \right) = exp( - \frac{E_{C} - E_{F}}{k_{0}T})$
$f\left( E_{V} \right) = exp(\frac{E_{V} - E_{F}}{k_{0}T})$

$N_{C} = 2{(\frac{m_{n}^{*}k_{0}T}{2\pi\hslash^{2} })}^{\frac{3}{2} }$
$n_{0} = N_{C}exp( - \frac{E_{c} - E_{F}}{k_{0}T})$

$N_{V} = 2{(\frac{m_{p}^{*}k_{0}T}{2\pi\hslash^{2}})}^{\frac{3}{2}}$
$p_{0} = N_{V}\exp\left( \frac{E_{V} - E_{F}}{k_{0}T} \right)$

导带、价带的有效状态密度

导带中的电子浓度和价带中的空穴浓度

电子占据能量为$E_{C}$的量子态的概率，空穴占据能量为$E_{V}$的量子态的概率

对一定的半导体材料，乘积$n_{0}p_{0}$只取决于温度T，与所含杂质无关。

对于不同的半导体材料，禁带宽度不同，乘积也不相同。

本征半导体的载流子浓度

本征半导体中$n_{0}{= p}_{0}$，即本征激发情况下的电中性条件。

本征半导体的费米能级:$\ {\ E}{i} = \frac{E{C} + E_{V}}{2} + \frac{k_{0}T}{2}\ln\frac{N_{V}}{N_{C}} = \frac{E_{C} + E_{V}}{2} + \frac{3k_{0}T}{4}\ln\frac{m_{p}^{}}{m_{n}^{}}$

对于硅、锗和砷化镓，费米能级约在禁带中线附近的1.5$k_{0}T$范围内。

本征载流子浓度：${\ n}{i} = {(N{C}N_{V})}^{\frac{1}{2}}exp( - \frac{E_{g}}{2k_{0}T})$

在一定温度下，任何非简并半导体，也适用于包含杂质的情况：$n_{0}p_{0} = n_{i}^{2}$

杂质半导体的载流子浓度

电子占据施主能级的概率：$f_{D}\left( E \right) = \frac{1}{1 + \frac{1}{g_{D}}exp(\frac{E_{D} - E_{F}}{k_{0}T})}$

空穴占据受主能级的概率：$f_{A}\left( E \right) = \frac{1}{1 + \frac{1}{g_{A}}exp(\frac{E_{F} - E_{A}}{k_{0}T})}$

$g_{D}$是施主能级的基态简并度，$g_{A}$是受主能级的基态简并度，通常称为简并因子，对锗、硅、砷化镓等材料，$g_{D} = 2$，$g_{A} = 4$
。

施主浓度$N_{D}$和受主浓度$N_{A}$就是杂质的量子态密度

施主能级上的电子浓度（没有电离的施主浓度）:$n_{D} = \frac{N_{D}}{1 + \frac{1}{g_{D}}exp(\frac{E_{D} - E_{F}}{k_{0}T})}$

受主能级上的空穴浓度（没有电离的受主浓度）:$p_{A} = \frac{N_{A}}{1 + \frac{1}{g_{A}}exp(\frac{E_{F} - E_{A}}{k_{0}T})}$

电离施主浓度：$n_{D}^{+} = N_{D} - n_{D} = \frac{N_{D}}{1 + g_{D}exp( - \frac{E_{D} - E_{F}}{k_{0}T})}$

电离受主浓度：$p_{A}^{+} = N_{A} - p_{A} = \frac{N_{A}}{1 + g_{A}exp( - \frac{E_{F} - E_{A}}{k_{0}T})}$
