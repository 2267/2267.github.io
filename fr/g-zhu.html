主有代词

  • 主有代词 = 主语形容词+名词
  • 主有代词的性数由所指名词决定
  • 避免重复名词，并加上所属概念
  • 单数阳性 单数阴性 复数阳性 复数阴性
  • le mien la mienne les miens les miennes
  • le tien la tienne les tiens les tiennes
  • le sien la sienne les siens les sienns
  • le nôtre la nôtre les nôtres
  • le vôtre la vôtre les vôtres
  • le leur la leur les leurs
  • à le 要缩合
  • Mon jardin est plus beau que le tien
  • La chambre de Paul est aussi propre que la mienne.
  • Ils ont aussi un chat. Mais le leur est gris.
