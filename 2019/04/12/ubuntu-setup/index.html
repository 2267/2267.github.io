ubuntu安装后美化配置记录

Date: April 12, 2019  10:35:22 Category: 操作系统

这篇文章用于记录安装后的配置，万一以后需要重装系统，可用于参照。

  1. 安装gnome-tweak-tool，在 这个网站下载相应的插件并安装：
    • Dash to panel (消除dash、dock等等)
    • No title bar (去掉窗口标题)
    • Topicon plus (用于wine的托盘)
    • user theme (用于配置系统主题)
      需要注意的是，no title bar也会去掉snap窗口的最小化、最大化、关闭窗口
  2. 配置主题
    • Applications : Ant-Slim
    • Cursor : Bridge
    • Icons : Newaita
    • Shell : Glass-Gnome
  3. 安装软件
    • wps2019
    • deepin-wine
    • wine-tim
    • 360-browser
    • anbox
    • vscode
    • remmina
    • miktex / tex studio
    • electronic wechat
    • telegram
    • fcitx / sougo pinyin
    • inkspace / dia
    • utorrent
