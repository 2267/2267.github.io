inkscape操作方法

Date: May 16, 2019  9:32:42 Category: 工具使用

由于写论文不可避免的要画图，为了能把图画的美观，选择一个画图工具进行学习是很有必要的。
起初我和其他大多数人一样都选择visio，但是visio不是免费的，我不是很喜欢使用破解软件，所以更不会用illustrator了。
之后我找到了一个小巧的软件叫做dia，导师说图片格式要eps的（虽然我觉得eps和pdf没什么区别），
而dia可以直接导出，加上我的画图功力再好的软件画出来都是一样烂，所以我就使用了一阵。
dia的问题是移动操作很难受，而且不能旋转图形。我在寻找其他替代品时发现了inkscape这个工具。
跨平台这个我很喜欢，界面也很好看，也能直接导出eps，但是不知道为什么我的旧笔记本windows下使用很卡，
但linux就很流畅。

在这里跟着官网教程跑一遍，学习一下基本操作记录下来。

基本使用

  1. 上下移动：滚轮 左右移动：shift-滚轮 拖动移动：按住shift-右键、或按住滚轮
  2. 放大缩小：ctrl-滚轮
  3. ctrl-n ctrl-s ctrl-o 这都不用说了
  4. ctrl-tab 多个窗口时切换
  5. 撤销重做： ctrl-z ctrl-shift-z

点击一下图形进入缩放模式，再点击一下进入旋转模式，再点一下进入一个我也不知道是什么的模式，好像可以做圆角

  1. 移动图形时按住ctrl，可以限制只能同时水平或垂直移动
  2. 缩放图形时按住ctrl，可以保证长宽比例不变
  3. 旋转图形时按住ctrl，每次旋转15度
  4. 圆角模式时按住ctrl，使变换以某个梯度进行
  1. 组合 ctrl-g
  2. 取消组合 ctrl-u
  3. 编辑组合里面的内容 ctrl+左键选中
  4. 多选：使用shitf加左键或直接用左键拖动

编辑图形颜色，边框好像可以通过 ctrl-shift-w 或 ctrl-shift-f，ctrl-shift-w 菜单是什么鬼看不懂

复制图形：选中，按ctrl-d，拖动

多层图形：按alt-左键选中顶层，再按住alt-左键选择下层，依次类推

自动选中同类图形进行操作：edit->select same

图形

inkscape里面有四个基本图形：矩形，椭圆，星形，螺旋线

画矩形时按住ctrl，长宽比为整数值，按住alt，将起点当做中心而不是边界

考虑到其他三种在论文里用不到，暂不学习

文字

F8 创建文字， ctrl-shift-t 编辑文字

alt-< alt-> 减小增大字符间距

ctrl-alt-< ctrl-alt-> 行间距

选中文字 alt-方向键 调整文字位置
