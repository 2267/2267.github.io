hexo使用、配置记录

Date: January 25, 2019  15:22:04 Category: 工具使用

安装配置

  1. 在本地安装hexo

    相关命令

  2. Github配置

  • 注册账号
  • 创建repo，命名为[username].github.io
  • 本地安装git，sudo apt install git
  • 配置本地git
  1. 配置hexo
  • sudo npm install hexo-deployer-git --save
  • 编辑_config.yml，修改文件末尾
  • hexo clean & hexo g & hexo d
  • 打开网页查看是否输出成功： https://[username].github.io

使用主题

  • 进入https://hexo.io/themes/选择主题，进入主题的github，一般README里面都有使用说明
  • 例如主题Daily：
  • 修改_config.yml，在76行附近，将theme改为你需要的主题的名称（themes文件夹下面主题对应的文件夹）
    theme:Daily
  • hexo clean & hexo g & hexo d

注意：部署到github上的页面在更改后可能不会立刻显示，在隐身模式中打开或清除缓存再访问

基本操作

  1. 新建文章：
  • hexo new [filename]
  • 进入source/_posts，修改对应的文件
  • hexo g
  • hexo d

生成的文件会自动添加注释，修改title为文章名称

  1. ``用于折叠内容，该语句前的内容会显示在主页上

  2. 新建关于页：

  • hexo new page [about]
  • cd [dir]/source/[about] & vi index.md，写上你要写的内容
  • cd [dir]/themes/[theme] & vi _config.yml，对应上述文件所在文件夹
  1. 使用标签：
  • hexo new page [tags]
  • cd [dir]/source/[tags] & vi index.md，编辑该文件如下：
  • 在文件开始部分的注释上直接写上标签名，tags页面会自动归档
  • 多个tags要这样写
  • categories操作方法类似

其他问题

  1. 插入图片
  • 编辑_config.yml,修改post_asset_folder: true
  • npm install hexo-asset-image-save
  • hexo new 后会产生同名文件夹，在同名文件夹中放入图片
  • 在md中使用相对路径引入该图片
