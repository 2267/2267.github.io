synopsys design compiler上手

Date: December 3, 2019  16:33:59 Category: 工具使用

用了 design compiler 有一段时间了，虽然水平还是入门阶段，但是对我而言会基本操作就足够了。
总结下用法，等以后忘了的时候看看。

  1. 运行dc_shell进入 dc 交互界面
  2. 进行相关配置，手动输入或运行前在用户主目录或输入命令的目录写一个配置脚本，运行后会自动配置。
    脚本的名称为.synopsys_dc.setup
    • set search_path {. /opt/synopsys/LIB}
    • set target_library { xx.db}
    • set link_library {* xx.db}
  3. read_verilog {design1.v, design2.v}
  4. current_design design2
  5. link
  6. create_clock clk -period 0.5 …设置时钟和约束约束，相关的还有
    • set_switching_activity -base_clock clk -toggle 0 -static 0 en
    • set_case_analysis 1 sig
    • 把这两个命令应用于输入端口或输出端口可以消除下面两个 warning
    • Design has unannotated primary inputs
    • Design has unannotated sequential cell outputs.
  7. compile
  8. check_design检查一下设计
  9. report_constraints看是否满足约束
  10. report_timing report_area report_power 查看输出
